O estilo é definido para o arquivo/página toda, não será aplicado a outros documentos/páginas.

Desvantagens:

  • O estilo ocupa muito mais linhas que o conteúdo.
  • Caso haja muitas páginas, terá que copiar o código para todas as outras.
