Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica, só tem forma)

Nessa frase, temos um termo em destaque usando a tag <strong> (dá sentido ao destaque, semântica)

Itálico / Ênfase

Nessa frase, temos um termo em itálico usando a tag <i> (não semântica)

Nessa frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark> (semântica)

E no outro parágrafo, temos outro texto marcado no final.

Texto Grande e Pequeno

Estamos criando um texto grande (queria um pouco maior, apenas alterando a forma, por isso se tornou obsoleto: não envolve semântica) e um texto pequeno (tem um sentido nos contratos, como texto menos importante, não alterando apenas a forma) nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e inficar que ele foi adicionado depois (semântica).

Existe também o sublinhado com a tag <u> (a formatação é a mesma que o ins, porém ela não é semântica, pois serve apenas para forma).
